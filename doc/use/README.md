# 目录结构

```
    bin---------------------------启动文件夹
    config------------------------配置文件
    controllers-------------------控制器
    doc---------------------------文档
    lib---------------------------引用文件
    log---------------------------日志文件
    middlewares-------------------中间件
    models------------------------数据层
    node_modules------------------依赖模块
    public------------------------静态文件
    routes------------------------路由
    services----------------------服务层
    .eslintrc.js------------------代码规范
    .eslintrcignore---------------代码规范
    .gitignore--------------------忽视文件配置
    app.js------------------------入口文件
    LICENSE-----------------------
    package.json------------------依赖文件
    package-lock.json-------------
    README.md---------------------介绍文档
```
